# Retail Revenue Analysis Dashboard (Power BI)

## Project Overview
This project analyzes an online retail dataset using Power BI to identify revenue trends, top-performing countries, and high-value customers. The goal is to generate business insights that support strategic expansion decisions.

## Tools Used
- Power BI
- Data Visualization
- Data Analysis

## Key Visualizations
1. Monthly Revenue Trend – Shows how revenue changes throughout the year.
2. Top 10 Countries by Revenue – Identifies countries generating the highest sales.
3. Top 10 Customers by Revenue – Highlights high-value customers contributing most to revenue.
4. Revenue Demand by Country (Excluding UK) – Visualizes global demand patterns using a map.

## Key Insights
- Revenue peaks toward the end of the year, indicating seasonal demand.
- Several European countries generate significant revenue outside the UK.
- A small number of customers contribute a large portion of total revenue.
- Geographic visualization helps identify potential markets for expansion.
